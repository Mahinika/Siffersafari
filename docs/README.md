## 📚 Diátaxis Framework

Dokumentationen följer **Diátaxis**, som delar in i 4 kategorier:

1. **Tutorials** — Lär dig genom att göra
2. **How-To Guides** — Lösa praktiska problem
3. **Reference** — Slå upp teknisk detalj
4. **Explanation** — Förstå varför

Se [diátaxis.fr](https://diataxis.fr/) för mer info.
