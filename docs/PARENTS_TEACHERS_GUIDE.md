# Guide – Föräldrar/Lärare (MVP)

## Översikt
Appen är en matte-träningsapp med profiler (barn), progression och ett föräldraläge med PIN.

## Profiler
- Skapa profil via **Inställningar → Skapa användare**.
- Välj aktiv profil i Inställningar om flera användare finns.
- Årskurs (Åk 1–9) påverkar svårighetsgrad.

## Föräldraläge (PIN)
- Öppna via lås-ikonen på startsidan.
- Ange PIN för att komma in.
- Inne i föräldraläget kan du:
  - Se översikt och senaste quiz.
  - Se enkel analys (svagaste områden + rekommendation).
  - Slå av/på räknesätt per användare (Addition/Subtraktion/Multiplikation/Division).
  - Byta PIN.

## Rekommenderat arbetsflöde
1. Skapa/byt till rätt profil.
2. Sätt årskurs.
3. Aktivera bara relevanta räknesätt i föräldraläge.
4. Låt barnet öva 5–10 minuter.
5. Följ upp statistik och rekommendationer.

## Vanliga frågor
**Var sparas data?**
- Lokalt på enheten (Hive). Ingen molnsynk i MVP.

**Fungerar appen offline?**
- Ja, grundflödet är validerat offline (lokal data + assets).
