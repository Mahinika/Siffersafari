# Mascot Walk Preview

Denna preview laddar referensfilen `artifacts/animation_reference/mascot_walk_cycle.json`.

## Lokal körning

1. Starta en enkel server i repo-roten, till exempel `py -m http.server 8765`.
2. Öppna sedan `http://127.0.0.1:8765/artifacts/animation_preview/mascot_walk_preview/` i webbläsaren.

Previewn är bara till för visuell validering utanför emulatorn. JSON-filen är arkiverad som previewmaterial under `artifacts/animation_reference/`.
